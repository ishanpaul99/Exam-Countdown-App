# Exam Countdown

Exam Countdown is a student productivity web application designed to help users track upcoming examinations through a real-time countdown interface. Users can enter an exam name, date, and time to view a live countdown showing the exact number of days, hours, minutes, and seconds remaining until the exam begins.

The application also includes color-coded status indicators that provide a quick visual representation of exam proximity, helping students stay aware of important deadlines and preparation timelines. Exam information is automatically saved locally, allowing users to continue tracking their countdown even after refreshing or reopening the application.

## Features

- Real-time exam countdown
- Exam date and time tracking
- Color-coded countdown status indicators
- Local storage support for saved exams
- Responsive design for desktop and mobile devices
- Simple and modern user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Browser Local Storage

## Development

The initial codebase was generated with the assistance of Cursor AI as part of an AI-assisted development workflow.

Further customizations were implemented manually, including countdown enhancements, exam time support, user interface refinements, logo integration, validation improvements, testing, and overall application polishing.
